> Prerequisites
To get started with this project, you will need to have Python 3.x installed on your computer. 
You will also need to have an API key from Alpha Vantage

> Installing Dependencies
To install the required packages, run the following command in your terminal:
pip install requests pandas matplotlib

> Running the Script
To run the script, simply execute the following command in your terminal:
python index.py


> Điều kiện tiên quyết
Để bắt đầu với dự án này, bạn sẽ cần cài đặt Python 3.x trên máy tính của mình.
Bạn cũng sẽ cần phải có khóa API từ Alpha Vantage

> Cài đặt phụ thuộc
Để cài đặt các gói yêu cầu, hãy chạy lệnh sau trong thiết bị đầu cuối của bạn:
Cài đặt PIP Yêu cầu Pandas Matplotlib

> Chạy tập lệnh
Để chạy tập lệnh, chỉ cần thực hiện lệnh sau trong thiết bị đầu cuối của bạn:
Python Index.py
