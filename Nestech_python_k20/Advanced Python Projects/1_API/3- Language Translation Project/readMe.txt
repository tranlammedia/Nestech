Prerequisites
Before you begin, you need to have the following:

An API key from NewsAPI. 
You can sign up for a free account and obtain an API key from https://newsapi.org/.


Installation
Install the NewsAPI client library by running the following command:
pip install newsapi-python



Điều kiện tiên quyết
Trước khi bạn bắt đầu, bạn cần phải có những điều sau:

Một khóa API từ Newsapi.
Bạn có thể đăng ký tài khoản miễn phí và lấy khóa API từ https://newsapi.org/.


Cài đặt
Cài đặt thư viện khách hàng Newsapi bằng cách chạy lệnh sau:
PIP Cài đặt Newsapi-Python
