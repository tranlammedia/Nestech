Required Libraries
To run this calculator GUI project, you need to have Python 3.x and the Tkinter library installed on your computer. Tkinter is a built-in library in Python, so you don't need to install it separately.

Running the Program
To run the calculator GUI, simply run the main.py file in your Python environment. This will open a window that displays the calculator GUI.

Using the Calculator
To use the calculator, simply click on the buttons to input numbers and operators. The calculator supports the following operations:

Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)

To perform a calculation, simply click on the '=' button. The result of the calculation will be displayed in the window.
